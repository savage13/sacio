void sac_header_read_v7 ( FILE *  fp,
sac *  s,
int *  nerr 
)

Read the sac header version 7

Arguments

  • nun - Negative file id number (yes, it is negative)
  • s - sac file structure to fill
  • nerr - Error reporting value

V7 of the header is a additional set of data at the end of the file following the data. This routine only reads the this "footer" of metadata, the v6 header I/O routines are still required
